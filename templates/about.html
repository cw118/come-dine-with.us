The Project and Vision

come-dine-with.us was our project for Bon Hacketit 2.0, a 48-hour long hackathon organized by MLH.

The pitch: a site for posting about food you'd like to share. Maybe you cooked too much and you're not sure what to do with the extras, or you don't feel like cooking and don't want to eat alone (or all of those combined)—come-dine-with.us has you covered!

Tech specs: a Flask app in the form of a responsive website. HTML and CSS (+ Bootstrap) on the frontend, with Python and some SQL on the server-side.

View the project on Devpost: https://devpost.com/software/come-dine-with-us

Additionally, you can find the source code at our GitHub repository here: https://github.com/piaoruilin/come-dine-with.us

What's Next

The time constraint and communication barriers due to time zones hindered our team's ability to complete the app's implementation, but here are some of the main features we'd originally planned to include:

  • User sign-up and login
  • Post filters: find destinations nearby, exclude ingredients based on applicable allergies, etc.
  • Mapping component: show the destination linked to a post and get directions
